# Polymarket UI SDK

Welcome to the Polymarket UI SDK documentation. This library provides a set of React components and hooks for building consistent and beautiful user interfaces.

## Installation

```bash
npm install polymarket-ui-sdk
```

## Usage

```jsx
import { Button, useTheme } from "polymarket-ui-sdk";

function App() {
  const { theme } = useTheme();

  return <Button variant="primary">Click me</Button>;
}
```

## Components

Our components are organized into the following categories:

- **Core** - Basic building blocks like Button, Input, etc.
- **Layout** - Components for structuring your application
- **Data Display** - Components for displaying data
- **Feedback** - Components for user feedback
- **Navigation** - Components for navigation

Each component is documented with:

- Props and their types
- Usage examples
- Best practices
- Accessibility considerations
